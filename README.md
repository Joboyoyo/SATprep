# SAT Reading Prep

A personal SAT Reading & Writing practice app with spaced repetition, stats tracking, and gamification.

## Features

- **75 SAT-style questions** across 5 categories and 3 difficulty levels
- **Spaced repetition review** — missed questions resurface at 1h, 1d, 3d, 1w intervals
- **Timed Test mode** — 27 questions in 32 minutes like the real SAT module
- **Weakness Drill** — auto-generated sessions targeting your weakest categories
- **Word Bank** — save vocabulary from questions, drill with spaced-repetition flashcards
- **Statistics** — accuracy by category/difficulty, contribution heatmap, predicted SAT score
- **Achievements** — 16 unlockable milestones
- **Daily streak tracking** with goal ring
- **Theme toggle** — Auto (time-based), Light, or Dark mode
- **Star questions** to revisit later
- **Right-click to strike through** obviously wrong answers

All data persists in your browser's localStorage.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The output will be in the `dist/` folder, ready for static hosting.
